# KeyFlow

macOS 키보드 유틸리티 앱. 글로벌 단축키로 앱 실행, 키보드 스크롤, 텍스트 자동 대치를 지원합니다.

## Download

[Latest Release](https://github.com/jeje-labs/Keyflow-Release/releases/latest) 에서 `KeyFlow.dmg`를 다운로드하세요.

## Features

### App Launcher
단축키로 앱을 즉시 실행/전환/숨기기. 한 번 더 누르면 토글됩니다.
- 카테고리별 자동 분류 (Developer Tools, Productivity 등)
- 단축키 템플릿 (⌘F1~F12)으로 빠른 설정

### Keyboard Scroll
Vim 스타일 키보드 스크롤. 마우스 없이 모든 앱에서 스크롤 가능합니다.

| 단축키 | 동작 |
|---|---|
| `⌃J` | 아래로 스크롤 |
| `⌃K` | 위로 스크롤 |
| `⌃D` | 반 페이지 아래로 |
| `⌃U` | 반 페이지 위로 |
| `⌃H` | 왼쪽 스크롤 |
| `⌃L` | 오른쪽 스크롤 |

### Text Expansion
한글 자모 조합을 입력하면 정형화된 문구로 자동 대치됩니다. CS(고객 서비스) 업무에 최적화되어 있습니다.

**사용법:**
1. 트리거 입력 (예: `ㅎㅂ`)
2. 미리보기 패널이 나타남
3. `Tab` 또는 `Enter`로 확정
4. 여러 후보가 있으면 `⌘1`~`⌘9`로 선택

**기본 제공 문구 예시:**

| 트리거 | 대치 텍스트 |
|---|---|
| `ㅎㅂ` | 안녕하세요, 고객님. 문의해 주셔서 감사합니다. 무엇을 도와드릴까요? |
| `ㅈㅅ` | 불편을 드려 대단히 죄송합니다, 고객님. 빠르게 확인하여 안내드리겠습니다. |
| `ㅂㅅ` | 주문하신 상품은 현재 배송 준비 중이며, 출고 완료 시 송장번호를 안내드리겠습니다. |
| `ㅎㅂㄹ` | 환불 요청이 접수되었습니다. 영업일 기준 3~5일 이내에 처리 완료됩니다. |
| `ㄱㅅ` | 소중한 시간 내어 문의해 주셔서 감사합니다. 좋은 하루 되세요! |

### Snippet Palette
`⌘F11`로 Spotlight 스타일 문구 검색 패널을 열어 등록된 문구를 검색/선택할 수 있습니다.

## 설치 방법

1. `KeyFlow.dmg` 다운로드
2. DMG 열기 → `KeyFlow.app`을 Applications 폴더로 드래그
3. 앱 실행
4. **접근성 권한 허용** (시스템 설정 → 개인정보 보호 및 보안 → 접근성 → KeyFlow 체크)

> 접근성 권한이 없으면 글로벌 단축키가 동작하지 않습니다.

## 주요 단축키

| 단축키 | 동작 |
|---|---|
| `⌘⇧K` | KeyFlow 설정 열기/닫기 |
| `⌘F11` | 문구 팔레트 열기/닫기 |

모든 단축키는 Settings에서 변경 가능합니다.

## 설정 백업

Settings → General → **전체 설정 내보내기/가져오기**로 바인딩 + 텍스트 대치를 JSON 파일로 백업/복원할 수 있습니다.

## Requirements

- macOS 14.0 이상
- 접근성 권한 필수
